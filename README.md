# Sigils of Elohim Solver
Intuitive but inefficient solver for the puzzle game Sigils of Elohim. Solves the harder problems in a few seconds.

## Usage
Start the program.  
Enter into stdin: \<rows> \<columns> \<pieces>

Example: "2 4 LL"  
produces the output:  
1112  
1222

Available pieces: LJIOSZT
